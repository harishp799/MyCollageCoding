# Email Spam Classifier

A Machine Learning project that classifies SMS/email messages as Spam or
Not Spam (Ham) using NLP techniques.

## Demo

Run locally: python -m streamlit run app.py

## Project Objective

Automatically detect spam messages using ML.

## Tech Stack

- Python
- Pandas, NumPy
- Scikit-learn
- TF-IDF
- Naive Bayes
- Streamlit

## Dataset

SMS Spam Collection Dataset (Kaggle/UCI)

## Workflow

1.  Data cleaning
2.  Text preprocessing
3.  TF-IDF vectorization
4.  Model training
5.  Evaluation
6.  Streamlit deployment

## Project Structure

spam-classifier/ ├── data/ ├── model/ ├── train.py ├── app.py ├──
requirements.txt └── README.md

## Run Steps

1.  pip install -r requirements.txt
2.  python train.py
3.  python -m streamlit run app.py

- Spam detection
